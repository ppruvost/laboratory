Analyse des résultats

Tu as maintenant observé la propagation du son dans deux situations différentes.

Observation 1 : dans l'air
Le son se propage depuis le diapason jusqu'au récepteur. Les vibrations produites par le diapason sont transmises par les molécules d'air.

Observation 2 : dans le vide
Aucun son ne peut être transmis car il n'y a aucune matière pour transporter les vibrations.

Interprétation :
Le son est une onde mécanique. Une onde mécanique a besoin d'un milieu matériel pour se propager.

Je retiens

✔ Le son se propage dans les solides, les liquides et les gaz.

✔ Le son ne se propage pas dans le vide.

✔ La présence d'un milieu matériel est indispensable à la propagation d'une onde sonore.
