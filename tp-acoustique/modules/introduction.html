Analyse des résultats

Tu as maintenant observé la propagation du son dans deux situations différentes.

Dans l'air : le son est transmis par les vibrations des molécules d'air. Les vibrations produites par le diapason se propagent jusqu'à l'oreille, qui reçoit alors le signal sonore.

Dans le vide : aucune particule n'est présente pour transmettre les vibrations. Le son ne peut donc pas se propager jusqu'au récepteur.

Conclusion : le son est une onde mécanique. Il a besoin d'un milieu matériel (solide, liquide ou gaz) pour se propager. Il ne peut pas se déplacer dans le vide.
