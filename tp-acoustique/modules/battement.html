TP - Battements sonores

Observer comment deux fréquences proches produisent des variations périodiques de l'intensité sonore appelées battements.

Fréquence de battement théorique : 2 Hz

Fréquence de battement observée : 2 Hz

À retenir

Lorsque deux fréquences proches sont émises simultanément :

  • la fréquence moyenne détermine la hauteur du son ;
  • la différence entre les deux fréquences produit les battements ;
  • plus les fréquences se rapprochent, plus les battements deviennent lents ;
  • lorsque les battements disparaissent, les fréquences sont identiques.
